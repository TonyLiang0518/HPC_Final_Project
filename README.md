# HPC_Final_Project

Author: He, Li, Liang

This repository is final project for NYU courant course High Performance Computing.

run `make` to compiler all files.

`./compare -n 10` get run time of recursion FFT, iteration FFT and their parallel version

`./fft_package -n 10` get run time of use fftw package

`./pde -n 5` get result of PDE using recusion FFT

`./fftw_pde -n 5` get result of PDE using fftw package
